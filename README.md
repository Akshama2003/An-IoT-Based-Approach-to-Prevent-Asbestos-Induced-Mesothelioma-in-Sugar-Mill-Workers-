# 🏭 An IoT-Based Approach to Prevent Asbestos-Induced Mesothelioma in Sugar Mill Workers


---

## 📌 Overview
This project presents an **IoT-based smart safety system** designed to prevent asbestos-induced diseases such as mesothelioma in sugar mill workers. It integrates **environmental monitoring, wearable health devices, cloud computing, and machine learning** to provide real-time alerts and predictive insights.

---

## 🎯 Objectives
- Real-time monitoring of air quality and harmful gases  
- Continuous tracking of worker health parameters  
- Predictive alerts using Machine Learning  
- Reduction of occupational health risks  

---

## 🚨 Problem Statement
Sugar mills often contain **aging asbestos infrastructure**, which releases harmful fibers.

### Challenges:
- ❌ No real-time hazard detection  
- ❌ No health monitoring of workers  
- ❌ Delayed response to unsafe conditions  

---

### 🔌 Hardware

- ADT7410 (Temperature Sensor)  
- BMP180 (Pressure Sensor)  

### 💻 Software
- Arduino IDE  
- Embedded C/C++  
- Firebase / Blynk  
- Machine Learning (SVM / Random Forest)

## 🔄 Working Methodology
1. Sensors collect environmental and health data  
2. ESP32 & Arduino process and transmit data  
3. Data stored on cloud (Firebase/Blynk)  
4. ML model predicts risk levels  
5. Alerts triggered via buzzer, LED, and dashboard
